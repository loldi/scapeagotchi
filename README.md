# Scapeagochi

A pixel art 2D game built with [Phaser 3](https://phaser.io/).

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Open the URL shown in the terminal (e.g. http://localhost:5173).

## Build

```bash
npm run build
```

Output is in `dist/`. Use `npm run preview` to serve the production build locally.
